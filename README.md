# 🌤️ Day & Night Theme — CSS Assignment

A simple web project demonstrating the three types of CSS styling.

## 📋 About
This project contains two sections on a single webpage:
- ☀️ **Day Theme** — Light colors, sky blue background, black text
- 🌙 **Night Theme** — Dark blue background, white text, starfield effect

## 🎯 CSS Methods Used
| Method | Where Used |
|--------|-----------|
| External CSS | Day section styles (style.css) |
| Internal CSS | Night section styles (inside `<style>` tag) |
| Inline CSS | Glowing star effect on Day heading |

## 📁 Files
- `index.html` — Main HTML structure
- `style.css` — External stylesheet
